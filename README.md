## Offline dictionary Api

### Api(Deno)

```sh
cd api4deno
# rename
mv .example.env .env
# dev
deno task dev
# build
deno task build
```

### Api(Go)

```sh
cd api4go
# dev
go run main.go
# build
go build -o endict main.go
```

### web(serverless)

- `pnpm install`
- `pnpm dev`
